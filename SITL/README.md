# SITL — окно 1

Сборка (при первом запуске) и старт ArduCopter SITL с параметрами полёта без GPS:

```bash
bash sitl_launch.sh
```

- `sitl_launch.sh` — запускает бинарь напрямую: `-w` (чистая память каждый
  запуск — воспроизводимость), `-I0` (порты: 5760 — мост, 5762 — Mission Planner),
  `--defaults vio_params.parm`;
- `vio_params.parm` — параметры автопилота: GPS выключен, позиция и курс —
  от VIO (extnav), высота — баро, компасы выключены;
- рабочие файлы SITL (eeprom, логи) складываются в подпапку `sitl_run/`.
